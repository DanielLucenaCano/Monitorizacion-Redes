Documentar de forma integral la implantación de una solución de monitorización de red en laboratorio virtual, con foco en trazabilidad, validación técnica y evidencias de ejecución.

## Criterios de trazabilidad

- Cada fase debe dejar constancia de comandos ejecutados.
- Cada decisión técnica debe estar justificada.
- Cada prueba debe incluir resultado esperado, resultado real y evidencia asociada.
- Cada avance relevante debe quedar registrado en Git con commits atómicos.
